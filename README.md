TaskFlow-Up is a task management web application that helps users organize and track their daily tasks efficiently.
- React / TypeScript
- Node.js
- Supabase
- Tailwind CSS
- Create, update, and delete tasks
- User authentication
- Real-time database updates
- Responsive UI
